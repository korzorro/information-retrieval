This category contains articles that incorporate a citation to the Encyclopedia Americana, which is in the public domain.

Wikipedia articles that summarise information from this source should add {{Cite Americana}} to the article.

Wikipedia articles that incorporate text from this source should add {{ Americana}} to the article.
